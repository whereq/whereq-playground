# Install nvm

```
curl https://raw.githubusercontent.com/creationix/nvm/master/install.sh | bash
source ~/.profile
nvm -version
nvm ls-remote
```

## More nvm commands
```
nvm --version
nvm ls
```

# Install node

```
nvm install v20.11.1
node -v
```

# Install yarn
```
npm install --global yarn
yarn -v

npm install --global cross-env
```
